Colaborativo

Somos un equipo abierto donde todos tenemos permiso para añadir y actualizar la información para poder tener y más actualizada lista de recursos posible, ¡únete!.

Open Source

Nuestros scripts están licenciados bajo GPL v3.0, para que puedas hacer lo que quieras con ellos. La plantilla usada para la web ("codrops-scribbler) está bajo MIT.

Reutilizable

Además de poder exportar los listados como CSV hemos creado un widget para facilitar que cualquiera pueda embeber los listados fácilmente en cualquier web.

Actualizado

Siempre es complicado mantener un listado de enlaces (porque tienden a romperse), por eso estamos creando un comprobador de enlaces rotos.

Core

El objetivo es poder derivar proyectos a partir de esta recopilación de enlaces, el widget es un ejemplo de ello. Descubre más en github.
